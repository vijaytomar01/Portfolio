# Vijay Pratap Tomar — Portfolio

A modern, responsive full-stack developer portfolio built with pure HTML, CSS, and JavaScript featuring:

- 🎨 Modern UI with glassmorphism design
- ✨ Smooth animations and scroll reveals
- 🎯 Custom cursor with smooth tracking
- ⚡ Type animation effect
- 📱 Fully responsive mobile design
- 🌙 Dark theme with accent colors

## Features

- **Hero Section** with profile card and animated introduction
- **About** section with personal info and background
- **Skills** Grid showcasing technical expertise
- **Experience** timeline with internship details
- **Projects** showcase with descriptions and tech stacks
- **Education** timeline
- **Certifications** display
- **Contact** section with form and social links

## File Structure

```
portfolio/
├── index.html      # Main HTML file
├── styles.css      # All CSS styles
├── script.js       # JavaScript functionality
├── profile.jpg     # Profile picture (add your image)
└── README.md       # This file
```

## Getting Started

1. Clone the repository
2. Add your profile image as `profile.jpg` in the root directory
3. Open `index.html` in a web browser

## Customization

- Edit `index.html` to update content
- Modify colors in `styles.css` (CSS variables in `:root`)
- Update social links and contact information
- Replace `profile.jpg` with your photo

## Technologies

- HTML5
- CSS3 (with Grid, Flexbox, CSS Variables)
- Vanilla JavaScript (ES6+)
